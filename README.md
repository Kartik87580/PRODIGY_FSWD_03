### 1. Clone the Repository

```bash
git clone https://github.com/Kartik87580/ PRODIGY_FSWD_03.git
````

---

### 2. Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file in the `backend/` folder:

```env
PORT=5000
MONGO_URI=your_mongo_uri
JWT_SECRET=your_secret_key
```

Start the server:

```bash
npm run dev
```

### 3. Frontend Setup

```bash
cd ../client
```
